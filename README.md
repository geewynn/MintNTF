# Getting Started
Requirements
1. Node
2. Yarn
3. Git

Clone this repository:

```bash
git clone https://github.com/geewynn/MintNTF.git

cd MintNTF
```

Install dependencies:

```bash

yarn install

```

Run the frontend:

```bash

yarn start

```

> in a second terminal window:

```bash
cd MintNTF
yarn chain

```

> in a third terminal window:

```bash
cd MintNTF
yarn deploy

```
